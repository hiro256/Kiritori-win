Kiritori-win
============

“Kiritori” is a simple screen capture tool.
“Kiritori” can capture your screen and display it in front of desktop.
“Kiritori” means “Cut” in Japanese.

[Kiritori for Macintosh](http://kiritori.ruhenheim.org)
